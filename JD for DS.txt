Job Title

Senior Software Engineer / Data & AI Platform Engineer

Job Summary

We are seeking a highly skilled and versatile Senior Software Engineer / Data & AI Platform Engineer to design, build, and scale cloud-native applications, data platforms, and AI-driven systems. The role combines backend and frontend engineering, data engineering, machine learning enablement, and DevOps practices to support mission-critical platforms in finance, healthcare, enterprise analytics, and intelligent automation environments.

This role requires strong expertise in Java, Python, cloud platforms, distributed systems, data pipelines, APIs, and ML integration, along with a proven ability to deliver scalable, secure, and high-performance systems in regulated and high-availability environments.

Core Responsibilities
Software Engineering and Full Stack Development

• Design and develop microservices-based systems using Java, Spring Boot, Python, and RESTful APIs
• Build and maintain frontend applications using React, Angular, or modern JavaScript frameworks
• Develop secure, scalable backend services handling high-volume transactions and real-time data flows
• Implement authentication and authorization using OAuth, JWT, and role-based access control
• Maintain clean architecture, design patterns, and SOLID principles across services

Data Engineering and Analytics Platforms

• Design and optimize ETL pipelines using Apache Spark, Airflow, AWS Glue, or Azure Data Factory
• Build and maintain data lakes and warehouses on Snowflake, Redshift, BigQuery, or Synapse
• Optimize SQL performance, indexing, and partitioning strategies for large-scale datasets
• Enable real-time data streaming using Kafka, Kinesis, or RabbitMQ
• Deliver BI-ready datasets for Power BI, Tableau, and executive dashboards

Machine Learning and AI Enablement

• Develop and deploy ML pipelines using Scikit-learn, TensorFlow, PyTorch, and statistical modeling techniques
• Integrate LLM and NLP workflows for document processing, resume screening, knowledge systems, and analytics automation
• Support predictive modeling, forecasting, anomaly detection, and classification systems
• Implement model monitoring, validation, and data quality checks

Cloud, DevOps, and Infrastructure

• Architect and deploy applications on AWS, Azure, or GCP using containerization and serverless services
• Build and maintain CI/CD pipelines using Jenkins, GitHub Actions, or GitLab CI
• Implement Infrastructure as Code using Terraform or CloudFormation
• Monitor systems using Prometheus, Grafana, ELK, and CloudWatch
• Enforce security, compliance, and governance standards across environments

Collaboration and Delivery

• Work in Agile and Scrum environments, participating in sprint planning, reviews, and retrospectives
• Collaborate with product managers, data scientists, analysts, and business stakeholders
• Provide mentorship, code reviews, and technical leadership
• Translate business requirements into scalable technical solutions

Required Technical Skills
Programming and Frameworks

Data and Analytics

• Apache Spark, Airflow, dbt, ETL pipelines, Data Warehousing
• PostgreSQL, MySQL, Oracle, Snowflake, Redshift, MongoDB, DynamoDB

Machine Learning and AI

Cloud and DevOps

• Jenkins, GitHub Actions, GitLab CI

Security and Compliance

Preferred Qualifications

• Experience building platforms in finance, healthcare, logistics, or enterprise SaaS environments
• Hands-on experience with streaming architectures and real-time analytics
• Experience with ML model deployment and MLOps workflows
• Background in data governance, auditability, and regulatory compliance systems
• Experience working with high-scale distributed systems and fault-tolerant architectures

Education

• Bachelor’s or Master’s degree in Computer Science, Data Science, Engineering, or related field

What Success Looks Like

• Delivers scalable, secure, and high-performance systems in production

• Maintains 95%+ code quality, test coverage, and deployment reliability
